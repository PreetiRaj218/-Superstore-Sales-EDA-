# -Superstore-Sales-EDA-
This project focuses on exploratory data analysis of the Superstore dataset to identify sales trends, profit distribution, customer behavior, and regional performance using Python (Pandas, Matplotlib).  The goal is to derive actionable business insights from raw data.

# 📊 Superstore Sales Analysis (EDA Project)

## 📌 Project Overview
This project performs Exploratory Data Analysis (EDA) on the Superstore dataset to uncover business insights related to sales, profit, customer behavior, and regional performance.

The goal is to transform raw sales data into actionable business insights.

---

## 🧹 Data Cleaning
- Converted `order_date` and `ship_date` into datetime format
- Checked and handled data types
- Verified missing values and duplicates

---

## 📊 Key Business Insights

### 1. Regional Performance
- Central region is the highest revenue generator.

### 2. Product Analysis
- Technology is the most profitable category.
- Furniture has the lowest profit margin.

### 3. Customer Analysis
- A small group of customers contributes significantly to total sales.

### 4. Sales Trend
- Sales show fluctuations over time indicating seasonal patterns.

---

## 📈 Tools Used

## 💡 Conclusion
This analysis helps identify key revenue drivers and areas of improvement, which can support better business decision-making.
